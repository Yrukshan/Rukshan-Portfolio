# Portfolio

Rukshan Portfolio is a modern, responsive, and visually engaging portfolio website built with Next.js, Tailwind CSS, and Framer Motion. It allows developers, designers, or tech enthusiasts to showcase their projects, skills, and blogs with smooth animations and an intuitive interface.


## Features

- Responsive design using Tailwind CSS
- Smooth animations with Framer Motion
- Dark mode support
- Modular and reusable components
- Easy to customize

## Tech Stack

- Frontend: Next.js (React framework)
- Styling: Tailwind CSS
- Animations: Framer Motion
- Deployment: Vercel (or any Node.js supported hosting)

## Installation

To get started with Devfolio, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone 
   ```

2. **Install the dependencies:**

   Make sure you have [Node.js](https://nodejs.org/) installed. Then, run the following command to install the necessary packages:

   ```bash
   npm install
   ```

3. **Run the development server:**

   Start the development server to view the project locally:

   ```bash
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

4. **Build for production:**

   To create an optimized production build, run:

   ```bash
   npm run build
   ```

5. **Start the production server:**

   After building the project, you can start the production server with:

   ```bash
   npm start
   ```
